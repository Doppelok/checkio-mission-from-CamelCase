All you have to do is wrap the code in a function expression and output the combined list (use "_" instead of spaces when combining). 
def from_camel_case(name):
    words = []
    for ch in name:
        if ch.isupper():
            words.append(ch.lower())
        else:
            words[-1] += ch
    return '_'.join(words)

Look at this solution. Quite an interesting solution using the f-line. Where every uppercase letter is replaced with a lowercase letter with "_", for example "D" will be replaced with "_d". And at the end, they simply cut off the extra "_", because the first letter was capital. 

def from_camel_case(name: str) -> str:
    res = ''
    for i in name:
        res += f"_{i.lower()}" if i.isupper() else i

    return res.lstrip("_")
